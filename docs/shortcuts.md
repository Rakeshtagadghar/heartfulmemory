# Editor Shortcuts

## Canvas / Node Shortcuts

- `Delete` / `Backspace`: Delete selected node(s) (locked nodes are skipped)
- `Ctrl/Cmd + D`: Duplicate selected node(s)
- `Ctrl/Cmd + C`: Copy selected node(s) to in-memory node clipboard
- `Ctrl/Cmd + V`: Paste node(s) with offset placement
- `Ctrl/Cmd + L`: Lock/unlock selected node(s)
- `Ctrl/Cmd + ]`: Bring selected node(s) forward
- `Ctrl/Cmd + [`: Send selected node(s) backward

## Selection / Editing

- `Shift + Click`: Add/remove node from current selection
- `Ctrl/Cmd + Click`: Add/remove node from current selection
- `Enter` on selected text box: Enter text edit mode
- `Escape` in crop mode: Cancel crop mode

## Notes

- Node shortcuts do not run while typing in text inputs, textareas, or text edit mode.
- Crop mode forces single selection while active.
