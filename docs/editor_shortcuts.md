# Editor Shortcuts (Sprint 5 v0)

## Supported Shortcuts
- `Ctrl/Cmd + B` → Bold
- `Ctrl/Cmd + I` → Italic

## Toolbar Actions (Mouse / Touch)
- `Bold`
- `Italic`
- `H2`
- `List`
- `Quote`

## Notes
- Sprint 5 uses a lightweight `contentEditable` text block implementation.
- TipTap/advanced editor behavior is deferred; storage remains block JSON-compatible.
- Image placeholder blocks are metadata-only in Sprint 5 (caption, placement, size).
