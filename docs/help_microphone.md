# Microphone Help

Short troubleshooting flow used by the wizard and studio voice UI:

1. Open the lock icon or site settings near the address bar.
2. Allow microphone access for this site.
3. Refresh the page.
4. If the mic is still unavailable, close other apps using the microphone.
5. If the wrong microphone is active, switch the input device in OS sound settings.
6. If voice still does not work, continue with typing instead.
