# DOCX Style Guide v1

## Fonts

| Element | Font | Size | Style |
|---------|------|------|-------|
| Title | Cambria | 28pt | Bold, centered |
| Subtitle | Cambria | 16pt | Italic, centered |
| Author | Cambria | 12pt | Normal, centered |
| Date | Cambria | 11pt | Normal, centered, gray |
| Chapter Heading | Cambria | 22pt | Bold |
| Body | Calibri | 11pt | Normal |
| Caption | Calibri | 9pt | Italic, gray |
| Credits heading | Cambria | 14pt | Bold |
| Credits body | Calibri | 9pt | Normal |

## Spacing

- Paragraph spacing: 6pt before, 6pt after
- Chapter heading: 24pt before, 12pt after
- After title page elements: 12pt spacing
- Line spacing: 1.15x (single with slight breathing room)

## Page Breaks

- Page break after title page
- Page break before each chapter heading (except first)

## Colors

- Body text: #1A1A1A (near-black)
- Captions/date: #666666 (medium gray)
- Headings: #1A1A1A

## Image Sizing

- Max width: 5.5 inches (within 1" margins on Letter)
- Aspect ratio preserved
- Centered horizontally
- 6pt spacing above and below
